# Portfolio de Projectes

Aplicació web per gestionar projectes desenvolupada amb Angular 19, Node.js, Express i MongoDB.

## Requisits previs

- Node.js instal·lat
- MongoDB corrent localment al port 27017

## Instal·lació i execució
```bash
npm i
npm run start
```

- Frontend: http://localhost:4200
- Backend: http://localhost:3700
